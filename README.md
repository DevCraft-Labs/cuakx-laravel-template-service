# Laravel Cuakx

A Laravel 13 microservice template for the Cuakx / Sumber Teknik platform. Use this as the starting point for every new service in the ecosystem.

## Stack

| Layer | Technology |
|---|---|
| Framework | Laravel 13 (PHP 8.3+) |
| Database | MySQL |
| Core library | [`cuakx/cuakx-core`](../Domains/cuakx-core) (local path) |
| Testing | PHPUnit 12 |

## Requirements

- PHP 8.3+
- Composer 2
- MySQL 8+
- Node.js 20+ / npm (for front-end assets, optional)

## Getting Started

### 1. Clone & install

```bash
git clone <repo-url> my-service
cd my-service
composer setup
```

The `composer setup` script will:
1. Install PHP dependencies
2. Copy `.env.example` → `.env` (if not present)
3. Generate the application key
4. Run database migrations
5. Install npm packages and build front-end assets

### 2. Configure environment

Edit `.env` and fill in your values:

```dotenv
APP_NAME=my-service
APP_ENV=local
APP_KEY=          # auto-generated by setup

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_database
DB_USERNAME=root
DB_PASSWORD=

REDIS_HOST=127.0.0.1
REDIS_PORT=6379
```

### 3. Run migrations

```bash
php artisan migrate
```

## Development

Start all dev processes (server, queue worker, log viewer, Vite HMR) in one command:

```bash
composer dev
```

Or run services individually:

```bash
php artisan serve          # HTTP server  → http://localhost:8000
php artisan queue:listen   # Queue worker
php artisan pail           # Log viewer
npm run dev                # Vite HMR
```

## Testing

```bash
composer test
```

This clears the config cache before running `php artisan test`.

## Using cuakx-core

`cuakx/cuakx-core` is available as a local Composer path dependency. Key utilities:

```php
use Cuakx\Core\Http\Controllers\BaseController;
use Cuakx\Core\DTO\BaseResponseDTO;
use Cuakx\Core\Utils\Console;
use Cuakx\Core\Utils\StringUtil;
use Cuakx\Core\Utils\CurrencyUtil;
use Cuakx\Core\Utils\Redis\RedisRepository;

// Standardised JSON response
return BaseResponseDTO::success('User fetched', $user);
return BaseResponseDTO::error('404', 'Not found');

// Validated controller
class UserController extends BaseController
{
    public function store(Request $request): JsonResponse
    {
        $this->baseValidator($request, [
            'name'  => 'required|string|max:255',
            'email' => 'required|email|unique:users',
        ]);

        // validation passed — proceed
    }
}

// Redis CRUD
$repo = new RedisRepository(prefix: 'user', defaultTtl: 3600);
$repo->set('42', $userData);
$user = $repo->get('42');
```

See [`Domains/cuakx-core/README.md`](../Domains/cuakx-core/README.md) for full documentation.
